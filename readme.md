解决国内网络访问问题

由于 raw.githubusercontent.com 在国内经常无法访问，直接把这个链接填进 Clash 可能会报错“获取失败”。建议使用加速代理前缀：



加速链接：在你的链接前加上 https://ghproxy.net/

最终示例：https://ghproxy.net/https://raw.githubusercontent.com/cym1754544/clash-config/main/AI.yaml
